Release Notes¶

NVIDIA VXGI 1.0

What’s New¶

Compared to beta version 0.91:

  • Added more control over the ambient occlusion mode (VXAO), including parameters such as ambient scale/bias and power exponent.
  • Added a diffuse tracing refinement pass that improves image quality in sparse tracing modes at a very small performance cost, especially when combined with temporal filtering.
  • Added support for using different diffuse tracing settings for a region of the screen marked in a stencil buffer.
  • Added an optional integrated screen-space ambient occlusion (SSAO) pass.
  • Added a simplified voxel invalidation mode that works well for applications with hundreds of moving objects.
  • Added support for Maxwell GPU features, such as FP16 atomics and pass-through geometry shaders, when running on D3D12.
  • Added getMinVoxelSizeAtPoint function that should be helpful for culling small objects during voxelization.
  • Added indirect irradiance clamping for cases when auto-normalization breaks or isn’t used.
  • Removed some experimental features that were not producing good enough results.
  • Improved performance of nearly every pass of the algorithm.
  • Many bugs fixed, especially when running on OpenGL.
